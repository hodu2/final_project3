조이름: 여행자바조

주제 : 국내 여행 숙박 예약 사이트 (관련 사이트 : 여기어때, 트리플, 에어비앤비)

기술 스택 :
  (Spring boot - java, Spring JPA - JPQL, QueryDSL, 
  (React or Thymeleaf - html, css, javascript), Oracle DB)

기능 :  

  1. 회원가입/로그인 (소셜 로그인 포함), 마이페이지 <반선우>
  : 회원가입 및 로그인: 소셜 로그인 (구글, 페이스북, 카카오 등) 지원.
  사용자 프로필 관리=마이페이지: 사진, 연락처, 예약 이력 등을 관리할 수 있는 페이지.
 (관리자/사용자 권한 분리: 예약 관리, 사용자 관리, 리뷰 관리 등.) 

  2. 숙소 검색 및 필터링(찜하기) + 메인페이지 <박선우>
  : 숙소 검색: 위치, 가격, 숙소 유형(호텔, 펜션, 아파트 등)으로 필터링.
  날짜 기반 예약 시스템: 예약 가능한 날짜를 체크하고, 달력에서 선택. 
  (가격대 및 평점 필터: 가격대나 이용자의 평점을 기준으로 검색 결과를 필터링).

  2-1. 지도 기반 위치 서비스
  숙소 위치 표시: 구글 맵 API를 활용해 숙소의 위치를 지도 위에 표시.
  근처 명소 안내: 숙소 주변의 관광지, 음식점 등 정보를 제공.
   -> 2번과 비슷한 기능

  3. 예약 및 결제 시스템 <김창범>
  : 예약 관리: 예약이 완료되면 자동으로 데이터베이스에 반영되고, 예약 가능한 날짜가 실시간으로 업데이트.
  -> 한국관광공사, 네이버,카카오 지도 api

     3-1. 결제 시스템: 토스나 카카오 api연동

    3-2. 예약 내역 및 취소 관리: 사용자가 본인의 예약 내역을 확인하고 취소할 수 있는 기능
   -> 이건 마이페이지에서 가능하도록

  4. 리뷰 시스템 (숙소 리뷰 작성, 평점 계산 등) + 숙소 상세페이지 <박보희>
  : 숙소 리뷰: 사용자가 숙소에 대한 리뷰를 남기고, 다른 사용자들이 이를 확인할 수 있는 시스템.
  리뷰 평점: 숙소의 평균 평점 계산 및 필터링 기능.
  리뷰 작성: 예약 후 리뷰 작성 기능 (텍스트, 사진 첨부 등).

  5. 숙소 등록 및 관리
  : 숙소 등록: 숙소 소유자가 숙소 정보를 입력하고 등록할 수 있는 관리 기능.
  숙소 사진 업로드: 숙소 사진을 업로드하고, 갤러리 형식으로 보여주는 기능.
  가격 및 정책 설정: 숙소의 가격 정책, 청소비, 예약 정책 등을 설정.)

  6. 어드민 대시보드 *
  : 예약 현황: 모든 예약 내역과 숙소 상태를 확인할 수 있는 대시보드.
  리뷰 관리: 사용자 리뷰를 승인하거나 차단할 수 있는 기능.
  숙소 관리: 숙소 등록, 수정, 삭제 기능.

  7. 추천 시스템 *
  : 렌터가 예약 및 사용자 선호도에 따른 여행지 추천


  알림 시스템 (푸시 알림, 예약 알림) *
  유동적으로 기능 추가및 삭제 예정.
